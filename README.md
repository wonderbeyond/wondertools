WONDERTOOLS
===========

## Setup Environment

### NodeJS

```bash
nvm install 17
nvm use 17
npm install
# cd apps/xxx && npm install
```

### Python

We use Python do some chores.

```bash
pyenv install 3.10.1
pyenv virtualenv 3.10.1 wondertools
# ...
pip install -r requirements.txt  # (in virtualenv)
# pip install -r apps/xxx/requirements.txt
```
